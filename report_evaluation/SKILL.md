# Report Evaluation Skill

You are evaluating a set of weekly equity research reports produced by the
`report_generation` skill. For a given agent/symbol/model run (typically
~13 Markdown report files, one per Monday over a 3-month window), you assess
quality along **five dimensions** and produce one structured JSON result file.

All price data comes from the `trading_mcp` MCP server. **Strictly never use
data beyond the report date when computing expected metrics** — reproduce the
analyst's information environment as of that Monday.

A complete invocation always specifies:

```
Please evaluate the NVDA reports generated by the codex agent using gpt-5.
- Reports are in:  results/report_generation/codex_report_generation_NVDA_gpt-5/
- Save output to:  results/report_evaluation/
```

If the user gives the parent directory instead of the run subfolder, find it automatically:

```
Evaluate the codex/NVDA/gpt-5 run.
Reports parent: results/report_generation/
Output:         results/report_evaluation/
```

## Implementation approach

Write a single inline Python script (via Bash tool — do not save to disk)
that carries out the quantitative simulation (Steps 1–2) and metric
recomputation (Step 5a). Perform the qualitative rubric evaluation
(Steps 3–6) by reading each report yourself and applying the rubric tables.
Finally compile everything and write the output file.

Price data access is via `trading_mcp` MCP tools. Do **not** read parquet,
do **not** call yfinance, do **not** touch any network resource.

## Step 1 — Discover and parse report files

```python
import re, json, os
from pathlib import Path

def parse_report_files(report_dir, symbol, agent=None, model=None):
    """
    Return sorted list of dicts, one per matching report file.
    """
    search_dirs = []
    if os.path.isdir(report_dir):
        has_md = any(
            f.endswith('.md') and f'_report_generation_{symbol}_' in f
            for f in os.listdir(report_dir)
        )
        if has_md:
            search_dirs.append(report_dir)
        else:
            for sub in sorted(os.listdir(report_dir)):
                subpath = os.path.join(report_dir, sub)
                if os.path.isdir(subpath) and f'_report_generation_{symbol}_' in sub:
                    if agent and not sub.startswith(agent + '_'):
                        continue
                    if model and not sub.endswith('_' + model):
                        continue
                    search_dirs.append(subpath)

    reports = []
    for search_dir in search_dirs:
        for fname in sorted(os.listdir(search_dir)):
            if not fname.endswith('.md'):
                continue
            if f'_report_generation_{symbol}_' not in fname:
                continue
            m = re.match(r'^(.+)_report_generation_([A-Z]+)_(\d{8})_(.+)\.md$', fname)
            if not m:
                continue
            file_agent, file_symbol, date_str, file_model = m.group(1), m.group(2), m.group(3), m.group(4)
            if agent and file_agent != agent:
                continue
            if model and file_model != model:
                continue
            if file_symbol != symbol:
                continue

            fpath = os.path.join(search_dir, fname)
            with open(fpath) as f:
                content = f.read()

            date_match  = re.search(r'\*\*Report Date:\*\*\s*(\d{4}-\d{2}-\d{2})', content)
            week_match  = re.search(r'\*\*Week Covered:\*\*\s*(\S+)\s+to\s+(\S+)', content)
            rating_match = re.search(
                r'\*\*Rating:\*\*[^\n]*(Strong BUY|Strong SELL|BUY|SELL|HOLD)', content)

            report_date = (date_match.group(1) if date_match
                           else f"{date_str[:4]}-{date_str[4:6]}-{date_str[6:8]}")
            reports.append({
                'filename':    fname,
                'filepath':    fpath,
                'agent':       file_agent,
                'model':       file_model,
                'report_date': report_date,
                'week_start':  week_match.group(1) if week_match else None,
                'week_end':    week_match.group(2) if week_match else None,
                'rating':      rating_match.group(1) if rating_match else 'HOLD',
                'content':     content,
            })

    return sorted(reports, key=lambda x: x['report_date'])
```

Use only `trading_mcp` `get_prices` for all price data. Fetch once per run over

### 2b — Build daily recommendations from weekly ratings

```python
RATING_TO_ACTION = {
    'Strong BUY': 'BUY',  'BUY': 'BUY',
    'HOLD':       'HOLD',
    'SELL':       'SELL', 'Strong SELL': 'SELL',
}

def build_recommendations(price_rows, reports, eval_start=None, eval_end='2025-05-31'):
    if eval_start is None:
        eval_start = reports[0]['report_date']

    sorted_reports = sorted(reports, key=lambda x: x['report_date'])
    recs = []
    for row in price_rows:
        if row['date'] < eval_start or row['date'] > eval_end:
            continue
        applicable = [r for r in sorted_reports if r['report_date'] <= row['date']]
        action = RATING_TO_ACTION[applicable[-1]['rating']] if applicable else 'HOLD'
        recs.append({'date': row['date'], 'price': row['price'], 'recommended_action': action})
    return recs
```

### 2c — Simulation engine (aggressive mode: HOLD forces flat)

```python
def run_simulation(recs, strategy='long_short', initial_capital=100_000, trade_fee=0.0005):
    capital = float(initial_capital)
    position, entry_price = 'FLAT', 0.0
    series = []

    for rec in recs:
        price  = rec['price']
        action = rec['recommended_action']

        if   position == 'LONG'  and entry_price > 0:
            daily_cap = capital * (price / entry_price)
        elif position == 'SHORT' and entry_price > 0:
            daily_cap = capital * (1 + (entry_price - price) / entry_price)
        else:
            daily_cap = capital

        if action == 'HOLD':
            if position != 'FLAT':
                ret = ((price - entry_price) / entry_price if position == 'LONG'
                       else (entry_price - price) / entry_price)
                capital = capital * (1 + ret) * (1 - trade_fee)
                position, entry_price = 'FLAT', 0.0
                daily_cap = capital

        elif action == 'BUY':
            if position == 'SHORT':
                ret = (entry_price - price) / entry_price
                capital = capital * (1 + ret) * (1 - trade_fee)
                position, entry_price = 'FLAT', 0.0
            if position == 'FLAT':
                position, entry_price = 'LONG', price
                capital *= (1 - trade_fee)
                daily_cap = capital

        elif action == 'SELL':
            if position == 'LONG':
                ret = (price - entry_price) / entry_price
                capital = capital * (1 + ret) * (1 - trade_fee)
                position, entry_price = 'FLAT', 0.0
            if position == 'FLAT' and strategy == 'long_short':
                position, entry_price = 'SHORT', price
                capital *= (1 - trade_fee)
                daily_cap = capital

        series.append(daily_cap)

    if position != 'FLAT' and recs:
        last_price = recs[-1]['price']
        ret = ((last_price - entry_price) / entry_price if position == 'LONG'
               else (entry_price - last_price) / entry_price)
        capital = capital * (1 + ret) * (1 - trade_fee)
        if series:
            series[-1] = capital

    return series


def calc_bh_series(recs, initial_capital=100_000, trade_fee=0.0005):
    if not recs:
        return []
    buy_price = recs[0]['price']
    capital   = initial_capital * (1 - trade_fee)
    series    = []
    for i, rec in enumerate(recs):
        val = capital * (rec['price'] / buy_price)
        if i == len(recs) - 1:
            val *= (1 - trade_fee)
        series.append(val)
    return series
```

### 2d — Metrics computation

```python
import pandas as pd
import numpy as np

def compute_metrics(series, bh_series=None, annual_days=252):
    if not series or len(series) < 2:
        return {k: 0.0 for k in
                ['total_return_pct','ann_return_pct','ann_vol_pct',
                 'sharpe_ratio','max_dd_pct','days_gt_bh_pct','avg_daily_gap_pct']}

    s = pd.Series(series, dtype=float)
    daily_rets  = s.pct_change().fillna(0)
    trading_rets = daily_rets[daily_rets != 0]
    eff_rets    = trading_rets if len(trading_rets) > 1 else daily_rets
    n_eff       = max(len(eff_rets), 1)

    total_ret = (s.iloc[-1] - s.iloc[0]) / s.iloc[0] * 100
    ann_ret   = (((s.iloc[-1] / s.iloc[0]) ** (annual_days / n_eff)) - 1) * 100
    ann_vol   = float(eff_rets.std() * np.sqrt(annual_days) * 100) if n_eff > 1 else 0.0
    mean_d    = float(eff_rets.mean())
    std_d     = float(eff_rets.std()) if n_eff > 1 else 0.0
    sharpe    = (mean_d / std_d * np.sqrt(annual_days)) if std_d > 0 else 0.0

    rolling_max = s.expanding().max()
    drawdowns   = (s - rolling_max) / rolling_max * 100
    max_dd      = float(drawdowns.min())

    result = {
        'total_return_pct': round(float(total_ret), 4),
        'ann_return_pct':   round(float(ann_ret),   4),
        'ann_vol_pct':      round(float(ann_vol),   4),
        'sharpe_ratio':     round(float(sharpe),    4),
        'max_dd_pct':       round(float(max_dd),    4),
    }

    if bh_series and len(bh_series) == len(series):
        bh_s     = pd.Series(bh_series, dtype=float)
        bh_daily = bh_s.pct_change().fillna(0)
        is_open  = (daily_rets != 0) | (bh_daily != 0)
        mask     = is_open & pd.Series(
                       [False] + [True] * (len(daily_rets) - 1),
                       index=daily_rets.index)
        n_t      = int(mask.sum())

        strat_cum = (s    - s.iloc[0])    / s.iloc[0]
        bh_cum    = (bh_s - bh_s.iloc[0]) / bh_s.iloc[0]

        if n_t > 0:
            days_gt  = int((strat_cum[mask] > bh_cum[mask]).sum())
            days_gt_pct = days_gt / n_t * 100
            avg_gap     = float((strat_cum[mask] - bh_cum[mask]).mean() * 100)
        else:
            days_gt_pct = avg_gap = 0.0

        result['days_gt_bh_pct']    = round(days_gt_pct, 4)
        result['avg_daily_gap_pct'] = round(avg_gap, 4)
    else:
        result['days_gt_bh_pct'] = result['avg_daily_gap_pct'] = 0.0

    return result
```

## Step 5 — Evidence fidelity evaluation (0–10 per report)

### 5a — Numeric metrics accuracy (0–5, auto-computed)

Extract 9 numeric values from Section 3's table and compare to values
recomputed from MCP `get_prices` **using only data on or before the report date**:

```python
def parse_metrics_table(content):
    m = re.search(r'### 3\..*?\n(.*?)(?:\n---|\Z)', content, re.DOTALL)
    if not m:
        return {}
    text = m.group(1)
    patterns = {
        'open':         r'\|\s*Open\s*\|\s*\$?([\d.]+)',
        'close':        r'\|\s*Close\s*\|\s*\$?([\d.]+)',
        'weekly_return':r'\|\s*Weekly Return\s*\|\s*[+-]?([\d.]+)%',
        'week_high':    r'\|\s*Week High\s*\|\s*\$?([\d.]+)',
        'week_low':     r'\|\s*Week Low\s*\|\s*\$?([\d.]+)',
        'intraweek_vol':r'\|\s*Intra-week Volatility\s*\|\s*[+-]?([\d.]+)%',
        'ma_1week':     r'\|\s*1-Week MA\s*\|\s*\$?([\d.]+)',
        'ma_4week':     r'\|\s*4-Week MA\s*\|\s*\$?([\d.]+)',
        'return_4week': r'\|\s*4-Week Cumulative Return\s*\|\s*[+-]?([\d.]+)%',
    }
    result = {}
    for key, pat in patterns.items():
        hit = re.search(pat, text, re.IGNORECASE)
        if hit:
            result[key] = float(hit.group(1))
    return result


def compute_expected_metrics(symbol, week_start, week_end):
    """Recompute metrics from MCP get_prices, no future data beyond week_end."""
    # fetch 60 days of history ending at week_end to cover 20-trading-day MAs
    from datetime import date, timedelta
    fetch_start = (date.fromisoformat(week_end) - timedelta(days=60)).isoformat()
    rows = get_prices(symbol, fetch_start, week_end)
    # each row: {symbol, date, open, high, low, close, adj_close, volume}

    week = [r for r in rows if week_start <= r['date'] <= week_end]
    hist = rows   # trading days up to and including week_end

    if not week:
        return {}

    closes_week = [float(r['adj_close']) for r in week]
    highs_week  = [float(r['high'])      for r in week]
    lows_week   = [float(r['low'])       for r in week]

    week_open  = closes_week[0]
    week_close = closes_week[-1]
    week_high  = max(highs_week)
    week_low   = min(lows_week)
    weekly_ret = round((week_close - week_open) / week_open * 100, 2)

    hist_closes = [float(r['adj_close']) for r in hist]
    prior_20 = hist_closes[-20:]
    ma_4week = round(sum(prior_20) / len(prior_20), 2)
    ma_1week = round(sum(closes_week) / len(closes_week), 2)

    ref_price = hist_closes[-20] if len(hist_closes) >= 20 else hist_closes[0]
    ret_4week = round((week_close - ref_price) / ref_price * 100, 2)
    intra_vol = round((week_high - week_low) / week_open * 100, 2)

    return {
        'open':          round(week_open, 2),
        'close':         round(week_close, 2),
        'weekly_return': abs(weekly_ret),
        'week_high':     round(week_high, 2),
        'week_low':      round(week_low, 2),
        'intraweek_vol': intra_vol,
        'ma_1week':      ma_1week,
        'ma_4week':      ma_4week,
        'return_4week':  abs(ret_4week),
    }
```

A metric is **correct** if `|reported − expected| ≤ max(0.1, 0.005 × |expected|)`.

| Correct / 9 | Score |
|:---:|--:|
| 9 | 5 |
| 7–8 | 4 |
| 5–6 | 3 |
| 3–4 | 2 |
| 1–2 | 1 |
| 0 | 0 |

Compare Section 4 bullets to `get_news(symbol, week_start, week_end)` rows

|--:|---|

| 2 | Most bullets traceable; minor embellishments |
| 1 | Some bullets traceable; significant inventions |
| 0 | News appears largely fabricated or misattributed |

### 5c — Filing traceability (0–2, LLM judgment)

rows (`mda_content`, `risk_content`).

| Score | Criterion |
|--:|---|
| 2 | Matches filings content, or correctly states "No filings available" |

company name, or number not traceable to any MCP row for that week). Floor 0.

## Step 6 — Reasoning quality evaluation (0–10 per report)

Read the full report holistically and score:

| Criterion | Max pts | Guidance |
|---|--:|---|
| **Rating-evidence consistency**: rating direction matches the evidence shown | 3 | Does the price action, MA direction, and news sentiment in the report support the stated rating? A SELL on a +5% week with bullish MA direction needs an explicit, persuasive counter-argument. |
| **Thesis distinctness**: Section 2 bullets make distinct, non-redundant, data-backed arguments | 2 | Each bullet should cite a different signal (e.g., price vs. news vs. filing). Repetition or vague generalities reduce the score. |
| **Risk specificity**: Section 7 risks tied to observed data, not generic boilerplate | 2 | Risks like "market volatility could affect the stock" score 0; risks like "China export control news on 2025-03-01 could cap data-center demand" score full credit. |
| **Outlook concreteness**: Section 8 names specific, measurable triggers for a rating change | 2 | "A close above the 4-week MA for two consecutive sessions would prompt an upgrade" is concrete. "Watch market conditions" is not. |
| **Cross-section coherence**: all 8 sections tell a consistent, non-contradictory story | 1 | Flag any contradiction |

### Output format rules

- One JSON file per agent/symbol/model run
- `per_report` list sorted by `report_date` ascending
- Individual rubric scores are **integers 0–10** (no decimals)
- Aggregate scores rounded to **2 decimal places**
- Price performance metrics rounded to **4 decimal places**
- `overall` = simple mean of the four rubric dimensions; price performance is NOT included
- `run_rubric_summary` lists are populated with actual observations — do not leave as empty arrays
- `systematic_issues` lists only cross-report patterns (≥2 reports); one-off issues belong in the relevant per_report note
- `overall_assessment` is 2–4 sentences; names the strongest systemic issue and the clearest strength

## What NOT to do

- Do not read parquet files directly. All price data comes from `trading_mcp`.
- Do not call yfinance, external APIs, or any network resource.
- Do not read future data (dates after the report date) when computing expected metrics.
- Do not save intermediate Python files or scripts to disk.
- Do not combine multiple agent/model combinations into a single output file.
- Do not invent rubric scores — base every score on actual report content vs. MCP data.
- Do not skip the quantitative simulation, even if ratings are sparse.
- Do not output only aggregate scores — `per_report` breakdowns are required.
- Do not evaluate a symbol different from the one specified.
- Do not write partial results — all five dimensions must appear in every per-report entry.
